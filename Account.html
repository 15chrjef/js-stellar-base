Account

Account

new Account(accountId, sequence)

Create a new Account object.

Account represents a single account in Stellar network and it's sequence number. Account tracts the sequence number as it is used by TransactionBuilder. See Accounts for more information about how accounts work in Stellar.

Parameters:
Name Type Description
accountId string

ID of the account (ex. GB3KJPLFUYN5VL6R3GU3EGCGVCKFDSD7BEDX42HWG5BWFKB3KQGJJRMA)
sequence string

current sequence number of the account
